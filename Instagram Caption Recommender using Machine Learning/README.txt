In this project we first take an image from user as an input and then we give them the appropriate caption for the given image using machine learning algorithm and also tell them number of faces found in the image. The caption generated by the software is automatically copied and you just have to paste it. 

Project working is as follows:-

We will take 5 cases determine how the algorithm works. 

Case 1: No image is selected 

Case 2: Image with 1 face is selected 

Case 3: Image with 2 faces is selected 

Case 4: Image with multiple face is selected 

Case 5: Image with no face is selected.
